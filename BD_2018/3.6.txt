Modelo E-R renta de videos

El siguiente caso de estudio describe los requisitos de datos de una empresa de alquiler películas de video. La empresa tiene diferentes sucursales en los Estados Unidos. Los datos almacenados para cada sucursal son la dirección de la sucursal (formada por la calle, ciudad, estado y código postal) y el número telefónico. Cada sucursal tiene un número de sucursal tiene una serie de empleados asignados entre los cuales hay un Gerente. El Gerente es responsable de la operación diaria de una sucursal determinada. Los datos que se almacenan sobre cada empleado son el nombre, la categoría y el salario. A cada empleado se le asigna un número de empleado que es univoco dentro de la empresa. Cada sucursal tiene una serie de películas de videos. Los datos que se almacenan sobre las películas son el número de catálogo, el número de película, el título, la categoría, el alquiler diario, el costo, el estado, el nombre de los actores principales y del director. El número de catálogo identifica unívocamente cada película de video. Si embargo, en la mayoría de los casos, hay varias copias de cada película en cada sucursal, por lo que las copias individuales se identifican utilizando el número de película. A cada película se le asigna una categoría, como por ejemplo comedia, drama, acción, infantil, terror o ciencia ficción. El estado indica si una copia específica de cada película esta disponible para alquilar. Antes de alquilar una película en la empresa, el cliente debe registrarse como cliente de la sucursal. Los datos que se almacenan sobre cada cliente son el nombre y apellido, la dirección y fecha en que se registro como cliente de la sucursal. A cada cliente se le da un número de cliente que es único entre las sucursales de la empresa. Una vez registrado el cliente puede alquilar películas libremente, hasta un máximo de 10 simultáneamente.  Los datos que se almacenan sobre cada película alquilada son el número de alquiler, el nombre completo y número del cliente, el número de película, el título y el alquiler diario, así como las fechas en que el video se alquiló y fue devuelto. El número de alquiler es univoco en toda la empresa.
